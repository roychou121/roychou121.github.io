前言

如何在 Ubuntu 18.04 上安裝 Anaconda3 ?

安裝

系統環境

  • OS：Ubuntu 18.04
  • Python：3.7.6
  • Anaconda3：2020.02

步驟

安裝使用工具 (選)

下載 Anaconda3

執行安裝檔

閱讀 License

一開始會跑出 license，按 Enter 看完

同意 License

輸入 yes 同意並繼續

儲存路徑

如果不改變位置，輸入 Enter 繼續執行

加入環境變數

輸入 yes ，把 anaconda 加入環境變數

修改權限

重啟環境變數

前言

Network FileSystem (NFS) 稱為網路文件系統，是一種分佈式文件系統協定，允許在伺服器上安裝遠端目錄，讓使用者可以管理不同位置的儲存空間。

NFS Server 需要先查詢硬碟狀態，確認是可被掛載的，如果是新硬碟，可參考此篇設定。

前言

Linux 中的 Parted 是一個用來管理磁碟分割區的工具，舉凡磁碟分割區的新增、刪除、大小變更等動作都可以用這個工具來處理。

fdisk 所能處理的磁碟容量上限是 2TB，若磁碟的容量大於 2TB 就無法使用。

前言

Royal OS System 是基於 Ubuntu Server & Desktop 客製化的系統。已安裝 AI 開發所需要的基本環境，可直接在系統上運行AI、機器學習、深度學習和分析應用程序所需的驅動程序。

介紹

特性

  • 大幅簡化建置流程
  • 定期釋出最新的版本
  • 已安裝 AI 開發所需的基本環境
  • 最佳化軟體相依性

硬體支援

  • CPU : Intel, AMD
  • NVIDIA GPU : Kepler, Maxwell, Pascal, Volta, Turing, Ampere

版本

  • Royal Server (ROYS) : v1.0.7
  • Royal Desktop (ROYD) : v1.0.2

安裝教學

以下分成 ROYS 及 ROYD 的實際安裝步驟及版本驗證，相對應的版本可至更新紀錄查詢

Royal Server (ROYS)

  1. 主機或伺服器接上 USB，並選擇 USB 開機

  2. GRUB 選單選擇 Install ROYS

  3. 等待錯誤檢查完成

  4. 選擇語系(建議 English)

  5. 選擇鍵盤模式(預設為 English US)

  6. 設定網路資訊或網路綁定

  7. 設定 Proxy

  8. 設定 Mirror 位置(預設即可)

  9. 選擇欲安裝的硬碟

  10. 確認將被格式化的硬碟及內容

  11. 設定帳號密碼等資訊

  12. 安裝 ssh (已安裝 可跳過)

  13. 等待安裝完成，完成後按下 Reboot Now

  14. 進入系統的歡迎畫面，可看到系統版本資訊

  15. 檢查 GPU 驅動

  16. 檢查 CUDA Toolkit 、 cuDNN 和 NCCL 版本

  17. 檢查 Docker 及 Python 版本

Royal Desktop (ROYD)

  1. 主機或伺服器接上 USB，並選擇 USB 開機

  2. GRUB 選單選擇 ROYD

  3. 等待錯誤檢查完成

  4. 選擇語系(建議 English) 並按下 Install ROYD

  5. 選擇鍵盤模式(預設為 English US)

  6. 安裝模式選擇(建議 Normal installation)

  7. 選擇 Erase disk and install ROYD

  8. 選擇欲安裝的硬碟

  9. 確認將被格式化的硬碟及內容

  10. 選擇時區

  11. 設定帳號密碼等資訊

  12. 等待安裝完成

  13. 按下 Restart Now 並取下 USB

  14. 進入系統後選擇 Settings -> Network ，開始設定網路資訊

  15. 檢查 GPU 驅動

  16. 檢查 CUDA Toolkit 、 cuDNN 和 NCCL 版本

  17. 檢查 Docker 及 Python 版本
